# Assets

This folder contains images used in the README and documentation.

## Expected Files

| File | Description |
|---|---|
| `glove.jpg` | Photo of the assembled Smart Glove prototype |
| `smart_home.jpg` | Photo of the assembled Smart Home control box |


## How to Add Your Photos

1. Place your image files in this folder
2. They will automatically appear in the README once uploaded to GitHub

> 💡 **Tip:** Keep image files under 5 MB each for fast loading on GitHub.
> If your poster is large (e.g. A0 print resolution), export a compressed version at ~1920px width.
